# Requirements

## Functional
- Wing sweep must vary between a defined minimum and maximum angle
- Sweep adjustment must be possible in flight
- Wing structure must remain continuous across sweep range
- Aircraft must be capable of stable glide at minimum sweep

## Design
- Mechanism must be diagrammable in plan and section
- Variable geometry must be visually and mechanically legible
- No reliance on active control laws for basic stability (conceptual)

## Documentation
- All geometry changes must be representable in standard diagrams
- All figures must comply with diagram_standards.md
